# دُكانك — متجر اشتراكات وألعاب رقمية

موقع متجر إلكتروني عرضي بدون بوابة دفع — عند الضغط على "تابع الطلب"، يتم تحويل العميل مباشرة إلى واتساب مع تفاصيل طلبه جاهزة للإرسال.

## الميزات
- كتالوج اشتراكات (Essential & Extra) لـ PS4 و PS5 بعدة مدد.
- كتالوج ألعاب مع خيارات PS4/PS5.
- سلة تسوق محلية (LocalStorage) لا تحتاج تسجيل دخول.
- إرسال الطلب عبر رابط `wa.me` مع الرسالة جاهزة.

## النشر على Netlify

### الطريقة ١ — السحب والإفلات (الأسهل)
1. شغّل أمر البناء محلياً:
   ```bash
   cd frontend
   yarn install
   yarn build
   ```
2. ادخل إلى [app.netlify.com/drop](https://app.netlify.com/drop)
3. اسحب مجلد `frontend/build` كاملاً وأفلته داخل الصفحة.
4. خلال ثوانٍ سيظهر رابط الموقع جاهزاً.

### الطريقة ٢ — ربط مع GitHub
1. ارفع المشروع على GitHub.
2. من Netlify اختر "Add new site" → "Import from GitHub".
3. ملف `netlify.toml` موجود مسبقاً ويحتوي على إعدادات البناء.

### تخصيص رقم الواتساب أو المنتجات
- رقم الواتساب: `frontend/src/data/products.js` — حقل `whatsapp` (بصيغة دولية بدون +).
- المنتجات والأسعار: نفس الملف، الكائنات `SUBSCRIPTIONS` و `GAMES`.

## التشغيل المحلي
```bash
cd frontend
yarn install
yarn start
```
الموقع سيعمل على http://localhost:3000
